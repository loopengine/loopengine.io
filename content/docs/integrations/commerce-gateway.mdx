## What it does

Commerce Gateway provides LLM-accessible commerce data, including product catalog, pricing, inventory, and supplier context through a unified API. AI actors in Loop Engine procurement and demand loops use this data as evidence before submitting governed recommendations.

- Without Commerce Gateway: AI actors lack structured commerce context.
- With Commerce Gateway: actors query commerce data, attach evidence, and submit decisions into runtime guards.

In **Commerce Chain** deployments, SCM and DCM procurement loops often combine Loop Engine governance with Commerce Gateway so AI actors can retrieve product, supplier, and inventory context before submitting governed transitions. See the [Commerce Gateway integration guide](https://commercechain.io/docs/integrations/commerce-gateway) on commercechain.io for the CCO-oriented setup and boundaries.

## Use case

SCM Procurement loop at `ANALYZING`:

1. AI actor queries Commerce Gateway for suppliers on `SKU-4892`
2. Gateway returns structured supplier + inventory data
3. Actor submits recommendation with attached evidence
4. Confidence guard and human approval gates evaluate transition

## Quick setup

1. **Install**

   ```bash
   npm install @loop-engine/adapter-commerce-gateway
   ```

2. **Create a `CommerceGatewayClient`**

   ```typescript
   import { CommerceGatewayClient } from "@loop-engine/adapter-commerce-gateway";

   const gateway = new CommerceGatewayClient({
     baseUrl: process.env.COMMERCE_GATEWAY_URL!,
     apiKey: process.env.COMMERCE_GATEWAY_API_KEY!
   });
   ```

3. **Query commerce context (typed client surface)**

   ```typescript
   // @no-typecheck
   // Illustrative: fetch inventory + price before you attach evidence to a governed transition
   const inv = await gateway.getInventory("SKU-4892");
   const price = await gateway.getCurrentPrice("SKU-4892");
   void { inv, price };
   ```

## Configuration reference

| Option | Type | Default | Description |
|---|---|---|---|
| `baseUrl` | `string` | required | Commerce Gateway base URL |
| `apiKey` | `string` | required | Commerce Gateway API key |
